# JavaScript Karşılama, Canlı Saat ve Gün Gösterimi

Bu proje, JavaScript kullanarak sayfa üzerinde **mevcut saati** ve **haftanın gününü** gerçek zamanlı olarak görüntüler.  
Saat, her saniye otomatik olarak güncellenir.

## Özellikler

- Sistem saatini görüntüler
- Haftanın gününü Türkçe olarak gösterir
- Otomatik olarak her saniye güncellenir

## Kullanılan Teknolojiler

- HTML
- CSS (Opsiyonel stil için)
- JavaScript (`Date`, `setInterval`)
